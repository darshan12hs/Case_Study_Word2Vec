# Case Study: Word2Vec

This repository contains multiple case studies and experiments using **Word2Vec**, a popular technique for learning word embeddings in Natural Language Processing (NLP).

## Overview

The goal of this project is to understand how Word2Vec captures semantic relationships between words and how these embeddings can be applied to real-world text data.

The repository includes experiments on different datasets such as product reviews, text corpora, and domain-specific data.


## Implementations

### 1. Amazon Reviews Analysis
- Applied Word2Vec on customer reviews  
- Learned word relationships from product feedback  
- Explored semantic similarity between words  

### 2. Game of Thrones Word2Vec
- Trained Word2Vec on GOT text data  
- Captured relationships between characters, places, and events  
- Visualized similarities and context-based meanings  

### 3. Cell Phones & Accessories Case Study
- Performed NLP preprocessing on product data  
- Built Word2Vec embeddings  
- Analyzed word relationships in domain-specific text  

### 4. Pre-trained Word2Vec Model
- Used pre-trained Word2Vec embeddings  
- Compared trained vs pre-trained embeddings  
- Tested similarity and vector operations  

## Key Concepts

- Word2Vec (CBOW & Skip-gram)  
- Word embeddings  
- Semantic similarity  
- Cosine similarity  
- Context-based learning
